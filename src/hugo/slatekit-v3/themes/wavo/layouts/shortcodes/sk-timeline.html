1 2015 Started off as an idea in 2015. Slate Kit was being built in-parallel with a mobile app/start-up. The idea was to keep the architecture side as decoupled from the business/mobile app as possible in Scala.
2 2016 Actively builing out the architecture components. Separating code into various projects for better organization
3 2017 Slate Kit almost fully extracted from a mobile startup and converted from Scala to Kotlin to allow for full-stack client ( Android ) and Server development and code reuse. This greatly reduced time, costs and made development for other projects much simpler using 1 language for Android and Server.
4 2018 Actively used in production and in beta apps. Dog-fooding this framework helped to polish up the components and discover and update use-cases. Slate Kit become a major effort and a cornerstone for some future SAAS product ideas.
5 2019 Emphasis on tooling, build automation, package distribution to BinTray, Unit-Tests and website updated with docs.
6 2020 Established stable, production ready, and well tested modules on both Server and Android, approach public release.
7 2021 Establishing a dual license for Slate Kit to allow for Apache 2.0 for majority of users. BSL for cloud providers. Publish Slate Kit on various channels to promote the project. A major Slate Kit Cloud Console to deploy and manage Slate Kit applications is in the works for 2021
