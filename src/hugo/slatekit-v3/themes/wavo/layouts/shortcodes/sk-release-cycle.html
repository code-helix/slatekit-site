Releases occur roughly every 2 weeks to 1 month for planned releases and uses Semantic Versioning in format **MAJOR.MINOR.PATCH**
Type Cycle Version Note(s)
Hot-Fix On Demand 1.0.2 Typically critical issues
Patches ~ 2 Weeks 1.0.3 Roughly every 2 weeks or sooner
Features ~ 1 Month 1.1.0 These are small or moderate changes and/or new features
Major ~ 3 Months 2.0.0 Major releases with new features and potentially breaking changes/ redesigns
